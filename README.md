# 🌱 OptiCrop

## 📌 Overview
OptiCrop is an AI-powered smart agriculture platform that helps farmers make better crop decisions using weather conditions, soil analysis, and machine learning predictions.

---
## ✨ Features
- 🌾 Crop Recommendation
- 🌦️ Weather Forecast
- 🌱 Soil Analysis
- 📊 Smart Dashboard
- 📍 User-Friendly Interface
- 🤖 AI-Based Predictions
---
## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Flask
- Machine Learning

## 🚀 Installation

```bash
git clone https://github.com/238x1a0444-paadhu/opticrop_project.git
cd opticrop_project
pip install -r requirements.txt
python app.py
```
---

## 💡 How It Works
1. Enter soil and environmental details.
2. The AI model processes the input.
3. The system predicts the most suitable crop.
4. Users receive smart farming recommendations.
---

## 🎯 Objectives
- Improve crop productivity.
- Support data-driven farming.
- Reduce farming risks.
